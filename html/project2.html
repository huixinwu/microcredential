🔙Dynamic Sales Website Project

This project consists of creating a sales website, with at least six products, applying all or almost of the web technology features we have learned in class. Those features include:

  • manipulation of HTML elements and how to design webpage’s layouts using semantic elements.
  • applying internal, external, and local links to texts and images
  • application of CSS properties and or Bootstrap classes to style different HTML elements such as text content, lists, navigations, images, and forms.
  • creating simple animation using CSS keyframe.
  • designing responsive webpages using CSS media query or Bootstrap.
  • creating modal windows using CSS attributes or Bootstrap.
  • manipulating different JavaScript or jQuery events to different elements in the website such as to open a modal window and evaluate forms.
  • planning to use JavaScript sessionStorage to transfer data within HTML pages of the same project.
  • creating a checkout form to send buyer’s name and email to a database
  • create a database table to collect buyer name and email
  • apply python framework, Flask, to transfer information from the sales website to the database.
