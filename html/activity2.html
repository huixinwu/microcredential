🔙Basic HTML elements
Using basic HTML elements to create a webpage with:
  • four images with the same height and width, title, and an external hyperlink. Hint: place the four images inside of the figure container. Also use figcaption
  • have internal links: links to three different places of the same webpage
  • have one description of definition list. Description title should be bold and italic.
  • have on table with background color in different cells, and bold title and subtitle.
  • have an unordered list.
  • have external links to Google maps or address
  • have a footer aligned to the center.
